
# Utilité

Le **Machine Learning (ML)** est un sous-domaine de l’intelligence artificielle qui permet aux machines **d’apprendre automatiquement à partir de données**, sans être explicitement programmées.

![AI](/learning/concepts/ai.png)

En résumé, un algorithme de machine learning n’est **pas programmé explicitement** pour chaque décision. Il apprend quelles actions sont optimales à partir des données existantes.

## Pourquoi utiliser le machine learning ?

Le machine learning est indispensable pour traiter :

* Des problèmes **trop complexes** pour être codés manuellement.
* Des volumes de données **trop importants** pour une analyse humaine.
* Des tâches qui nécessitent de la **prédiction ou de l’automatisation**.

:::info Exemples 
filtrage spam, lecture d’adresses manuscrites par les services postaux, reconnaissance de l’écriture, prédiction financière, etc.
:::

## L’importance des données

L’efficacité du machine learning dépend **principalement de la qualité des données** :

* Nettoyer et organiser les données est souvent la tâche la plus longue et la plus critique.
* La mise en œuvre de l’algorithme est généralement simple grâce aux bibliothèques Python comme **Scikit-learn**.
* Développer de nouveaux algorithmes est rarement nécessaire : la plupart des solutions sont déjà disponibles et bien documentées.

> *Un modèle n’est jamais meilleur que ses données.*
