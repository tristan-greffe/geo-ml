# Reconnaissance Faciale

L'objectif est de créer un scrip python qui 
1. enregistre un visage via la camera
2. essaye de se connecter factissmeent via des images de test ou par la camera

DATA: https://opencv.org/
https://github.com/opencv/opencv



HOW TO USE
1. ## Install Project Dependencies

The project uses Python 3.11, managed through `environment.yml`.

Create the conda environment and install all dependencies:

```bash
conda env create -f environment.yml
```

Activate the environment:

```bash
conda activate facial-recognition
```

Desactivate:
conda deactivate
