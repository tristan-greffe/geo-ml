# GeoML

**GeoML** est un projet appliqué : entraîner des modèles de machine learning sur des données géospatiales réelles, et afficher les résultats sur une carte interactive.

<img src="/geoml_pipeline.svg" style="display: block; margin: 0 auto;width: 100%; height: auto;">

## Fonctionnalités

**🌳 Détection d'arbres** : Entoure les couronnes d'arbres individuels sur la carte et fournit des statistiques : nombre d'arbres, superficie estimée et type d'arbre.

**🏠 Détection de bâtiments** : Entoure les bâtiments sur la carte et fournit des statistiques : nombre de bâtiments, superficie estimée et type de bâtiment.
