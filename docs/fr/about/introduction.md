# GeoML

**GeoML** est un projet d'apprentissage appliqué : construire des modèles d'intelligence artificielle pour analyser des données géospatiales, et les intégrer dans des applications cartographiques interactives.

Chaque projet couvre le pipeline complet — **dataset → modèle → API → carte**.

## Projets

| Projet | Technique | État |
|---|---|---|
| Détection de couronnes d'arbres | Segmentation sémantique (U-Net) | En cours |
| Détection de bâtiments | Détection d'objets | Prévu |
| Classification d'occupation du sol | Classification multi-classes | Prévu |

## Stack

Python · PyTorch · FastAPI · MapLibre GL JS · VitePress
