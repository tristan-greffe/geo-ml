# GeoML

**GeoML** is a hands-on learning project applying machine learning to GIS (Geographic Information Systems).

The goal: build complete end-to-end projects — from training a model on geospatial data to visualizing results on an interactive map.

Each project covers the full pipeline: **dataset → model → API → map**.

## Projects

| Project | Technique | Status |
|---|---|---|
| Tree crown detection | Semantic segmentation (U-Net) | In progress |
| Building detection | Object detection | Planned |
| Land cover classification | Multi-class classification | Planned |

## Stack

Python · PyTorch · FastAPI · MapLibre GL JS · VitePress
