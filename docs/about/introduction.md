# GeoML

**GeoML** is a hands-on project applying machine learning to GIS (Geographic Information Systems) : from training a model on real aerial imagery to displaying results on an interactive map.

<img src="/geoml_pipeline.svg" style="display: block; margin: 0 auto;width: 100%; height: auto;">

## Features

**🌳 Tree detection** : Outlines individual tree crowns on the map and provides statistics: count, estimated area, and tree type.

**🏠 Building detection** : Outlines buildings on the map and provides statistics: count, estimated area, and building type.
